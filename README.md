# Image Collage
Image Collage Maker is a lightweight web app designed for creating simple yet beautiful collages using your own images. 


## Technologies Used
- Next.js
- React
- CSS for styling


## Installation

```bash
git clone https://github.com/gvrv03/Image-Collage-.git
npm install 
npm run dev
```
